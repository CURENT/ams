=============
Input formats
=============

AMS currently supports the following input formats:

- ``.xlsx``: Excel spreadsheet file with AMS data
- ``.raw``: PSS/E RAW formats
- ``.m``: MATPOWER format
- ``.py``: PYPOWER format
